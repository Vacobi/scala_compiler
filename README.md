# scala_compiler

Компилятор языка **Scala** (версия 2.13) в байт-код виртуальной машины **JVM**, с
поддержкой части функционала исходного языка.

Учебный проект по дисциплине «Теория формальных языков и методов трансляции» (ТЯП),
ВолгГТУ, группа ПрИн-467. Авторы: Д.А. Смирнов, Р.В. Юрасов.

## Стадии трансляции

Исходный код проходит через следующие этапы:

1. **Лексический анализ** — разбиение текста на токены (Flex).
2. **Синтаксический анализ** — построение AST по грамматике языка (Bison).
3. **Нормализация дерева** — преобразование AST.
4. **Семантический анализ** — несколько проходов по AST: сбор символов (классы, поля,
   методы), привязка областей видимости и разрешение имён, проверка типов и корректности
   объявлений (наследование, `override`, модификаторы), заполнение таблиц символов и
   пула констант.
5. **Генерация кода** — эмиссия байт-кода `.class` и упаковка в исполняемый JAR.

## Поддерживаемый функционал

✅ — реализовано, ❌ — не реализовано.

| Возможность | Статус | Примечание                                                                                           |
|---|:---:|------------------------------------------------------------------------------------------------------|
| Классы | ✅ | в т.ч. `final`, `abstract`                                                                           |
| Одиночное наследование | ✅ | `extends` с передачей аргументов в конструктор родителя, напр. `class Son(n: String) extends Person(n)` |
| Первичный конструктор | ✅ | параметры-поля в заголовке класса, напр. `class Point(val x: Int, var y: Int)`                       |
| Вторичные конструкторы | ✅ | `def this(...) = this(...)`                                                                          |
| Абстрактные методы и атрибуты | ✅ | переопределение через `override`, динамическая диспетчеризация                                       |
| Модификаторы доступа | ✅ | `private`, `protected` (по умолчанию public)                                                         |
| Перегрузка методов | ✅ | по типам/числу параметров                                                                            |
| Перегрузка операторов | ✅ | метод-оператор, напр. `def +(o: Vec): Vec`                                                           |
| Методы `def` | ✅ | один список параметров, тип возвращаемого значения обязателен, дефолтные параметры не поддерживаются |
| Переменные `val` / `var` | ✅ | `val` — неизменяемая (повторное присваивание запрещено), `var` — изменяемая; тип обязателен, вывода типов нет |
| Перекрытие имён в скоупах | ✅ | переменная во вложенном скоупе перекрывает одноимённую из внешнего (возможно с другим типом), напр. `val x: Int = 4; { val x: String = "a" } // здесь x снова Int` |
| Встроенные типы | ✅ | `Int`, `Double`, `String`, `Char`, `Boolean`, `Unit`                                                 |
| Массивы `Array[T]` | ✅ | в т.ч. многомерные                                                                                   |
| Ветвления | ✅ | `if`, `if-else`, `if-else if`                                                                        |
| Циклы | ✅ | `while`, `do-while`, `for` (допускается лишь один генератор в объявлении)                            |
| `return` | ✅ |                                                                                                      |
| Операторы | ✅ | арифметические, сравнения, логические/битовые, унарные — с приоритетами                              |
| Расширенные идентификаторы | ✅ | символьные имена методов (`+`, `<=`, `+------+`)                                                     |
| Литералы | ✅ | целые (dec/hex/bin), `Double`, char, string, `true`/`false`/`null`                                   |
| Escape-последовательности | ✅ |                                                                                                      |
| Опциональная `;` | ✅ | допускается перенос строки в качестве разделителя                                                    |
| Комментарии | ✅ | однострочные `//` и многострочные `/* */`                                                            |
| Консольный ввод-вывод | ✅ | `Predef.println`/`print`, `StdIn.readInt`/…                                                          |
| `object` / companion object | ❌ |                                                                                                      |
| `trait`, миксы типов через `with` | ❌ | только одиночное наследование                                                                        |
| `enum` | ❌ |                                                                                                      |
| Pattern matching (`match`/`case`) | ❌ |                                                                                                      |
| Исключения (`try`/`catch`/`throw`) | ❌ |                                                                                                      |
| Интерполяция строк (`s"..."`) | ❌ |                                                                                                      |
| Лямбды / функциональные литералы | ❌ |                                                                                                      |
| Дженерики / параметры типов | ❌ | кроме встроенного `Array[T]`                                                                         |
| Вывод типов | ❌ | явное указание типов обязательно                                                                     |
| `import` / `package` | ❌ |                                                                                                      |
